Сценарий использования 1 - Заполнение ряда

Фигурка летит до тех пор, пока не наткнётся на другую фигурку либо на дно стакана. Если при этом заполнился горизонтальный ряд, он пропадает.

function removeLine(n) {

    var x, y;
    for (y = n; y >= 0; --y) {
        for (x = 0; x < nx; ++x)
            setBlock(x, y, (y == 0) ? null : getBlock(x, y - 1));
    }
}

Сценарий использования 2 - Сброс фигурки

Можно «сбрасывать» фигурку, то есть ускорять её падение, когда уже решено, куда фигурка должна упасть 

function dropPiece() {
    eachblock(current.type, current.x, current.y, current.dir, function(x, y) {
        setBlock(x, y, current.type);
    });
}

Сценарий использования 3 - Поворот

В полёте игрок может поворачивать фигурку на 90°/

function rotate() {
    var newdir = (current.dir == DIR.MAX ? DIR.MIN : current.dir + 1);
    if (unoccupied(current.type, current.x, current.y, newdir)) {
        current.dir = newdir;
        invalidate();
    }
}
